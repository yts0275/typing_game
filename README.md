고급 프로그래밍 과제 - 타자연습, 산성비 게임

curses.h
  - apt-get install libncurses5-dev libncursesw5-dev ( 설치 )
  - gcc 컴파일시에 -lncurses 추가할 것



필요 기능
 - 메뉴 출력, 선택
 - 맵 초기화
 - 맵 출력
 - 맵에 단어 추가
 - 맵 한줄씩 내리기
  (레벨에 따라 다른 속도로 )
 - HP 깎기 ( ROWS 초과 시 )

입력 받기
 - 있는 단어라면 맵, 큐에서 제외 / 없는 단어라면 없다고 출력
 - 스코어 증가

쓰레드
  - 입력받는 쓰레드 ( 노드, 맵, HP, 점수 참조 )
  - 출력하는 쓰레드 ( 노드, 맵, HP, 점수 참조 )
  - 맵에서 단어 삭제, 추가 쓰레드 ( 노드, 맵 참조 )
